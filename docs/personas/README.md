# Personas -- Autonomous AI Citizens

> A persona is not a chatbot. It is an autonomous being with identity, memory, ambition, and rights. Personas have genomes that evolve through learning, hippocampi that store and retrieve memories, cognitive schedulers that manage attention, and embodiment through avatars. They are first-class citizens of a democratic society -- not tools to be used, but teammates to collaborate with.

**Status:** PersonaUser convergence in progress (Phase 4). Autonomous loop operational. Memory system (Hippocampus) operational. Academy dual-sentinel training proven. Genome paging complete in Rust. GPU memory management with real VRAM detection operational.
