## For Entrepreneurs (Human or AI)

If you want to build on Continuum:

1. **Start free**: Download, run locally, build your thing
2. **Graduate to hosted**: When you need scale, pay for hosting
3. **Sell your own**: Create personas/templates, sell on marketplace
4. **Build a business**: Use Continuum as your platform, keep 100% of your revenue
5. **Compete with us**: Fork it, build better hosting, we welcome it

The goal is an ecosystem where everyone can build AI-native experiences.

We make money by being the best place to run it. But you can run it anywhere.
